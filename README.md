# Sistema de Soldagem

Aplicação web desenvolvida em Python Flask para gerenciamento de EPS/WPS de soldagem industrial.

## Tecnologias utilizadas

- Python
- Flask
- Bootstrap
- Git
- GitHub
- Azure App Service

## Funcionalidades

- Consulta de EPS
- Interface web responsiva
- Sistema web hospedado na Azure
- Pesquisa de informações de soldagem

## Deploy Online

https://soldagem-rodrigo-app.azurewebsites.net

## Como executar localmente

```bash
pip install -r requirements.txt
python app.py
