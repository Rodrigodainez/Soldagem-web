# 🔧 Sistema de Soldagem

Aplicação web desenvolvida em **Python Flask** para gerenciamento de **EPS / WPS de soldagem industrial**, com autenticação integrada ao **Microsoft Entra ID (Azure AD)**.

---

## 🚀 Tecnologias utilizadas

- Python
- Flask
- Bootstrap
- Git / GitHub
- Azure App Service
- Microsoft Entra ID (Azure AD)

---

## ✅ Funcionalidades

- 🔍 Consulta de EPS / WPS
- ➕ Cadastro, edição e exclusão de registros
- 🌐 Interface web responsiva
- 🔐 Autenticação corporativa via Azure
- 👤 Controle de acesso por usuário
- ☁️ Deploy em nuvem (Azure App Service)

---

## 🔐 Segurança

O sistema utiliza autenticação via **Microsoft Entra ID** e validação de usuários no backend.

- Acesso restrito por lista de usuários autorizados
- Processamento seguro de claims do Azure
- Proteção de todas as APIs
